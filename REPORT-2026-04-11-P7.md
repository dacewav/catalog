## ✅ Sistema Hover Avanzado (4 propiedades nuevas)

Implementación completa en admin, store CSS y store JS.

### Propiedades nuevas:
| Propiedad | Input | Rango | Descripción |
|---|---|---|---|
| Blur hover | Slider | 0-10px | Aplica blur a la tarjeta al hover |
| Blur hermanos | Slider | 0-8px | Las demás tarjetas se blurrean (`:has()` CSS) |
| Hue rotate hover | Slider | 0-360° | Rota matiz de colores al hover |
| Opacidad hover | Slider | 0.1-1 | Cambia opacidad de la tarjeta al hover |
| Animación hover | Toggle + Select + Slider | 20 tipos | Activa una animación SOLO al hacer hover |

### CSS (store-styles.css):
```css
.beat-card.has-hover-fx:hover {
  filter: brightness(var(--hov-bright,1)) saturate(var(--hov-sat,1)) 
          blur(var(--hov-blur,0px)) hue-rotate(var(--hov-hue,0deg));
  opacity: var(--hov-opacity,1);
}

.beat-grid:has(.beat-card.has-hover-fx:hover) .beat-card.has-hover-fx:not(:hover) {
  filter: blur(var(--hov-sib-blur,0px));
}

.beat-card.has-hover-anim:hover {
  animation: var(--hov-anim-name) var(--hov-anim-dur,1s) ease-in-out infinite;
}
```
