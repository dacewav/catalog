```bash

# 6. Subir a GitHub

git push origin main   # GitHub Pages despliega automáticamente
```

## Notas importantes

- **GitHub Pages** sirve desde branch `main` en `https://dacewav.github.io/catalog/`
- **Firebase config** está hardcodeado en `src/admin/config.js` (FC object). NO es un secreto — es público (client-side).
- **R2 Worker** se configura desde el admin panel (Settings → Cloudflare R2). Los tokens se guardan en Firebase `settings/r2Config`.
- **Whitelist de admin** está en `src/admin/firebase-init.js` (DEFAULT_ALLOWED_EMAILS) Y en Firebase `adminWhitelist`.
- **El token de GitHub usado en esta sesión ya fue revocado** — necesitarás generar uno nuevo para hacer push.
- **store-app.js** (58KB original) se mantiene como backup. El store usa `dist/store-app.js` (48KB bundle).
- **Circular deps** entre core.js ↔ colors.js (esbuild las resuelve, pero al refactorizar core.js hay que tener cuidado).
