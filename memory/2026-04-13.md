## Bug #1: Panel overlap — FIX APLICADO (necesita verificación en browser)

**Status:** Fix 4 aplicado — pendiente verificación en browser.

**Fix 4 (esta sesión):** Cambiado `showEt()` en `src/admin/nav.js` para usar `style.display` inline en vez de CSS classes. Esto gana sobre CUALQUIER regla CSS porque inline styles tienen máxima especificidad.

```js
// ANTES (class toggle — podía ser overriden por CSS):
p.classList.add('on');

// DESPUÉS (inline style — siempre gana):
p.style.display = 'block';  // para mostrar
p.style.display = 'none';   // para ocultar
```

También se aplicó inline style en `openEditor()` al resetear tabs.

**Cache busting:** build.js ahora agrega `?v=HASH` automáticamente a CSS/JS refs en HTML. Esto resuelve el problema de Cloudflare/browser cache.

2. Abrir beat editor → click en tab "Plataformas"

## Patrón para dependencias circulares (documentado)
Para extraer funciones de core.js que dependen de otras funciones de core.js:
```js
// En el módulo nuevo:
let _dep1, _dep2;
export function setXxxDeps({ dep1, dep2 }) { _dep1 = dep1; _dep2 = dep2; }

// En core.js:
import { setXxxDeps } from './xxx.js';
// Después de que las funciones existan:
setXxxDeps({ dep1, dep2 });
```
