## P8: Animation Intensity System + Per-Type Sub-Settings + Play Fix

### ✅ Play button no longer restarts animations
- `_updateIcons()` in `player.js` no longer calls `renderAll()` — uses targeted DOM class toggling
- New `_updatePlayingState()` uses `data-id` to match cards
- Play-pulse ring animation (600ms) for non-disruptive feedback
- `data-id` attribute added to beat card elements

### ✅ Animation intensity slider (0–100%)
- New `--anim-int` CSS variable, 0–1 range
- All 25+ keyframes use `calc()` with `--anim-int` for scalable magnitude
- Slider "Intensidad" in admin animation section

### ✅ Per-animation sub-settings panels
- Dynamic panel "⚙️ Ajustes específicos" shows controls per animation type
- Map: holograma→colors/bright/sat, brillo→min/max, glitch→X/Y/dist, translate→X/Y, neon→min/max opacity, parpadeo→min/max, rotate→angle/scale, scale→min/max/opacity, shake→X/Y, cambio-color→hue/sat
- `_toggleAnimSubsettings(type)` in beats.js

### ✅ All CSS keyframes updated to use per-type CSS vars
- `--anim-holo-bright-min/max`, `--anim-holo-sat-min/max`, `--anim-holo-glow`, `--anim-holo-blur`
- `--anim-brillo-min/max`, `--anim-glitch-x/y/rot`, `--anim-neon-min/max/bright`
- `--anim-parpadeo-min/max`, `--anim-rotate-angle/scale`, `--anim-scale-min/max/opacity`
- `--anim-translate-x/y/rot`, `--anim-shake-x/y`, `--anim-cs-hue-start/end/sat`
