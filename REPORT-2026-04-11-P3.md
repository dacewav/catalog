### 4. ✅ Slider Stability Fix
**Bug:** Moving any slider caused particles to go wild, unrelated elements updating.

**Root cause:** Each message type (theme, settings, emojis, floating) had its own independent debounce in the iframe handler. Admin broadcast was 80ms — too fast for the iframe to batch.

**Fixes:**
- Admin: broadcast debounce 80ms → 150ms
- Iframe: single shared debounce with per-type JSON dedup (skip identical data)
- `renderFloating()`: skip rebuild if data unchanged
- Commit: `d657b84`
